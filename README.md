# Project_P&G
This project is a FastAPI application connected to a SQL Server database.  
It manages manufacturing quality data for a shampoo production process.

## Technologies

- Python
- FastAPI
- SQL Server
- SQLAlchemy
- PyODBC
- Pytest
- GitHub Actions

## Database Structure

The database contains the following tables:

- Identifiers
- Countries
- ConsumerUnits
- Ownership
- Relationships
- Characteristics
- IdentifierCharacteristics
